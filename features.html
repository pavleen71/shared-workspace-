Features of CoWrkr

Main Features

  • User Registration and Authentication
  • Role-based Access Control (Owner, Coworker)
  • Listing and Managing Properties
  • Listing and Managing Workspaces
  • Searching and Filtering Workspaces
  • Viewing Workspace Details
  • Requesting Contact Information of Workspace Owners

Bonus Features

  • Adding Photos to Properties/Workspaces
  • Rating Workspaces on a Scale
  • Rating Coworkers on a Scale
  • Sorting Workspaces by Ratings, Availability, or Price
  • Writing and Viewing Workspace Reviews

Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express
  • Database: MongoDB (or any preferred database)

Minimum Viable Product (MVP)

  • User Signup and Login
  • Listing Properties and Workspaces
  • Searching and Filtering Workspaces
  • Requesting Contact Information of Workspace Owners
